###### --Proje Açıklaması--



a) Programın Amacı:



Bu program kullanıcının terminalden girdiği verileri kullanarak bir web sayfası (index.html dosyası) oluşturur.



b) Kullanılan Kütüphaneler:



Bu programda Python'un kendi kütüphanesinden başka bir kütüphane kullanılmamıştır.



c) Programın Algoritması:



1. Adım(Veri Alma): input() fonksiyonları kullanılarak kullanıcıdan Ad, Soyad, Bölüm, Dersler gibi veriler sırasıyla istenir.

2. Adım(Veri İşleme): Kullanıcının girdiği veriler birer değişkene atanır ve dersler kısmında liste replace() metodu ile liste oluşturulur.

3. Adım(HTML Kodu): Kullanıcının girdiği veriler f-string metodu ile HTML kodunun içine yerleştirilir, HTML kodunun içinde CSS kodları gömülüdür.

4. Adım(Dosya Oluşturup Yazma): Yeni bir index.html dosyası oluşturulur, içine HTML kodumuz yazılır ve işlem tamamlanır.



d) Program Nasıl Çalıştırılır ?:



1. Python yüklü bir bilgisayarda kodun içinde olduğu klasöre gidin.
2. Terminali açın.
3. (python (BMG101_proje).py) yazın.
4. Sorulan soruları cevaplayın.
5. Program bittiğinde aynı klasörde oluşan index.html dosyasına çift tıklayarak açın.
